poseidon-home
=====================

### Usage

```
npm install
bower install
npm start
```


### Pushing

To push to your desired branch and gh-pages.

```
npm run build
git add .
git commit -m 'Your message here.'
git push origin [branch]
git subtree push --prefix dist origin gh-pages
```
